# Iris Flower Classification

This project uses the Iris dataset from scikit-learn to classify flowers into three species: setosa, versicolor, and virginica.

## What I Built
- Loaded and explored the Iris dataset.
- Trained a decision tree model.
- Trained a logistic regression model.
- Compared both models using accuracy scores and confusion matrices.

## Results
- Decision Tree Accuracy: 1.000
- Logistic Regression Accuracy: 1.000

## Conclusion
Both models performed perfectly on this train-test split. The confusion matrices show no classification errors.
